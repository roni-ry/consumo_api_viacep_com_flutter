# Aplicativo de Consulta de CEP e CRUD de Endereços

Um aplicativo Flutter que consome a API do ViaCEP para consultar informações de CEP e usa o Back4App como banco de dados para implementar operações CRUD para entidades de endereços.

## Visão Geral

Este aplicativo é uma demonstração de como criar um aplicativo Flutter que permite aos usuários consultar informações de CEP usando a API do ViaCEP e realizar operações CRUD em endereços, que são armazenados no Back4App. O aplicativo oferece as seguintes funcionalidades:

- Consultar informações de CEP a partir da API do ViaCEP.
- Adicionar, visualizar, editar e excluir endereços da lista.
- Armazenamento de dados de endereço no Back4App.
- Suporte a autenticação de usuário (opcional).
